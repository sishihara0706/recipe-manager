# Recipe Manager

Java + Gradle で作成した、コンソール用のレシピ管理ツールです。  
現在はシンプルな「Hello Recipe Manager!」の表示のみですが、  
今後、レシピの追加・一覧表示・検索などの機能を実装していく予定です。

## 特徴

- Java 21 / Gradle を利用したシンプルなコンソールアプリ
- Windows 環境での開発を想定
- Gradle Wrapper を利用しているので、追加で Gradle をインストールする必要なし
- 2人での共同開発を前提（Git / GitHub 管理）

---

## 開発環境

- OS: Windows 10 / 11
- JDK: 21 以上
- Git
- エディタ: Visual Studio Code  
  - Extension Pack for Java  
  - Gradle for Java
